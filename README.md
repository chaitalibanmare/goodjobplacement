# GoodJobPlacement Frontend

This is a simple React + Vite frontend scaffold for GoodJobPlacement.com.

Run locally:

```bash
npm install
npm run dev
```

Open: http://localhost:5173

This project includes:
- Navbar with Vacancies, Courses, Features, Placement Activity, Community
- User and Admin login modals (demo only)
- Contact details in footer

Customize components in `src/App.jsx` and styles in `src/styles.css`.
